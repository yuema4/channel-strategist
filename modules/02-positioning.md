# 模块2：定赛道

> 目的：选对方向。基于诊断结果，决定主赛道+子集+保留项。

---

## 工作流

```
[入口] diagnosis完成 + 用户说"找赛道"/"我该学谁"
   ↓
[判断] 用户是否已有大方向？
   ├── ❌ 没有 → Step 2.0：Ikigai自测（新手模式）
   └── ✅ 有 → 跳过Ikigai，直接进入 Step 2.1
   ↓
[Step 2.1] 五重过滤找对标（内置）
   ↓
[Step 2.2] 3把筛子（手动）
   ↓
[Step 2.3] 飞轮模型搭建（手动）
   ↓
[输出] 一句话定位 + 飞轮图 + 18月节奏
```

---

## Step 2.0：Ikigai自测（新手模式）

### 什么时候触发

用户说以下任一关键词时，**先走Ikigai再谈赛道**：
- "我不知道做什么"
- "帮我找到我的方向"
- "我不确定"
- "新手小白"
- "完全没有头绪"
- "什么赛道适合我"
- "我适合做什么"

### Ikigai是什么

> Ikigai = 你热爱的 × 你擅长的 × 世界需要的 × 有人付钱的

```
          你热爱的
          (Passion)
              ↑
    使命       |       激情
  (Mission)   |   (Delight)
      ←-------+-------→
 世界需要的   |   你擅长的
(World Needs) |  (Skills)
      ←-------+-------→
    变现       |       专业
  (Market)    |   (Vocation)
              ↓
          有人付钱的
          (Market Value)
              ‖
           Ikigai
        (你的频道定位)
```

### 4圈自测（AI引导用户逐一回答）

**引导方式**：AI逐圈提问，每圈用户回答后，AI做归纳，然后进入下一圈。**不要一次性抛出4个问题**。

---

#### 圈1：你热爱什么？（Passion）

> "有什么话题，你愿意不拿钱也聊3个小时？"

**AI问用户的4个问题**（渐进式，先泛后精）：

| 顺序 | 问题 | 目的 |
|:---|:---|:---|
| 1 | 你空闲时间**天然会刷什么内容**？（小红书/B站/抖音上，不看会觉得少了什么） | 找到"天然注意力" |
| 2 | 你身边人**经常问你什么建议**？（朋友/同事/家人反复问你哪类问题） | 找到"被认可的价值" |
| 3 | 过去半年，你**花了最多时间/钱在上面的事**是什么？ | 找到"真实投入" |
| 4 | 如果只剩一年生命，你**最想分享给世界的一件事**是什么？ | 找到"核心驱动力" |

**AI归纳格式**：
```
你的"热爱圈"关键线索：
- 天然注意力：{X}
- 被认可的价值：{Y}
- 真实投入：{Z}
- 核心驱动力：{W}

归纳：你似乎天然被「{X}」吸引，而且身边人认可你在「{Y}」上的判断。
```

---

#### 圈2：你擅长什么？（Skills）

> "有什么事情，你做起来比别人轻松、别人觉得难？"

**AI问用户的4个问题**：

| 顺序 | 问题 | 目的 |
|:---|:---|:---|
| 1 | 你的**职业技能**是什么？（做过什么工作，接受过什么训练） | 可迁移的硬技能 |
| 2 | 别人**夸你什么**最多？（不是客套话，是反复出现的关键词） | 被外部验证的软技能 |
| 3 | 什么事情你做完了**不觉得累**，别人做完了累瘫？ | 隐藏的天赋点 |
| 4 | 你有什么**奇怪的资源**？（设备/人脉/经验/身份，别人没有的） | 差异化壁垒 |

**AI归纳格式**：
```
你的"擅长圈"关键线索：
- 硬技能：{X}
- 软技能：{Y}
- 天赋点：{Z}
- 壁垒资源：{W}

归纳：你的核心能力是「{X} + {Y}」，而且你有{W}这个别人没有的资源。
```

---

#### 圈3：世界需要什么？（World Needs）

> "市场上有什么需求还没被满足？"

**AI帮用户做的扫描**（主动搜索，不是问用户）：**

| 扫描项 | 方法 |
|:---|:---|
| 平台缺口 | 在小红书/B站搜索圈1+圈2的关键词，看头部内容的评论区**高频提问** |
| 对标空白 | 搜索相关博主，看他们**不做什么**（评论区求推荐/求教程的内容） |
| 趋势交叉 | 把圈1和圈2画交叉，问"这两个领域的交集，平台上有人做吗？" |

**AI归纳格式**：
```
你的"世界需要圈"关键线索：
- 平台缺口：用户在搜「{X}」但找不到好的内容
- 对标空白：现有博主在做「{Y}」，但没人在做「{Z}」
- 趋势交叉：「{A} × {B}」这个交叉点在平台上几乎是空白

归纳：市场上最需要的是「{一句话描述}」这个方向的内容。
```

---

#### 圈4：什么能变现？（Market）

> "这个方向有人愿意付钱吗？"

**AI帮用户做的检查**：**

| 检查项 | 方法 |
|:---|:---|
| 广告主是否存在 | 搜索同类博主的**商单内容**（看谁在投广告） |
| 用户愿不愿意付费 | 搜索相关关键词的**知识付费/课程/社群** |
| 客单价区间 | 估算：广告单价 × 商单频率 + 自有产品单价 × 转化率 |

**AI归纳格式**：
```
你的"变现圈"关键线索：
- 广告主：{X类型品牌}在投这个赛道的广告
- 用户付费：有{Y}门相关课程/社群在卖
- 预估变现天花板：月入{Z} - {W}

归纳：这个赛道有「{一句话}」的变现路径。
```

---

### Ikigai自测输出格式

**AI在4圈全部走完后，输出一份Ikigai合成报告**：

```markdown
# 你的Ikigai频道定位

## 四圈合成

| 圈 | 关键线索 |
|:---|:---|
| 热爱 | {一句话} |
| 擅长 | {一句话} |
| 世界需要 | {一句话} |
| 能变现 | {一句话} |

## 交叉点发现

- **热爱 × 擅长**：「{方向A}」—— 你能持续产出
- **擅长 × 变现**：「{方向B}」—— 能赚钱但可能不够热爱
- **变现 × 世界需要**：「{方向C}」—— 市场缺口但可能你不够擅长
- **世界需要 × 热爱**：「{方向D}」—— 有使命感但变现不确定

## 🎯 Ikigai中心（优先尝试）

> 「{最终方向}」

这是四圈重叠最多的方向。建议作为**主赛道候选**进入下一步（3把筛子验证）。

## ⚠️ Ikigai边界提醒

- 四圈完全重叠是理想状态，**现实中通常只有2-3圈重叠**
- 如果"变现圈"暂时空白 → 不代表不能做，代表需要**先涨粉再变现**
- 如果"擅长圈"空白 → 可以先做"热爱×世界需要"方向，**边做边学**
```

---

### Ikigai → 3把筛子衔接

Ikigai输出后，**直接进入Step 2.2的3把筛子**（不用再走五重过滤）：

```
Ikigai中心方向 → 筛子1（资源匹配）→ 筛子2（市场稀缺）→ 筛子3（战略一致）→ 主赛道
```

**如果Ikigai给出了2-3个候选方向**，则先走五重过滤找对标，再进3把筛子。

### Ikigai网页版（Capsule风格）

**在Ikigai合成报告文本版输出后**，AI必须问用户：

> "要不要生成网页版的Ikigai报告？（Capsule风格）"

**触发规则**：
- 只问一次，不等用户主动提
- 用户说"要""好""生成"→ 生成HTML并浏览器打开
- 用户说"不用""不需要"→ 跳过，直接进入3把筛子

**Capsule风格特征**：
- 底色 `#FAFAF8`，纯白卡片，暖灰边框
- 字体：Inter（英文/数字）+ Noto Serif SC（中文）
- 四色区分四圈：热爱=陶土橙、擅长=鼠尾草绿、世界需要=淡蓝、变现=琥珀黄
- Ikigai中心=深蓝紫胶囊卡片
- 全部圆角（10-32px），无阴影
- 胶囊/药丸形标签（capsule tag）
- 响应式，移动端断点600px

**HTML生成方式**：

AI需构造以下JSON数据，然后用Python脚本写入HTML文件：

```python
data = {

**生成后**：启动本地 HTTP 服务器，用浏览器打开生成的 HTML 文件。

## Step 2.1：五重过滤找对标（内置）

### 核心信念

> "0 到 1 阶段，模仿是正确答案。大部分'做自己'的人不敢挑战模仿的难度。"

**4 条信条**：
1. **排除自我是决策加速器** — 所有关于"我适不适合"的讨论都是噪音。只问：能干就执行，不能干就换。
2. **0 到 1 阶段，模仿优先** — 差异化是后话，先活下来。
3. **模仿的颗粒度决定模仿的质量** — 要对标到具体的获客方式、标题结构、发布时间，不是模糊的"感觉"。
4. **能变现是唯一标准** — 粉丝多 ≠ 赚钱。选对标只看一条：他能稳定变现吗？

---

### Phase 1：搞清楚用户现在的状态

**AI第一句话**：
> "你现在在做什么方向？如果还没开始，你想做什么？"

关键判断：
- 用户已有账号 → 帮他对标同赛道更赚钱的博主
- 用户还没开始 → 从 Ikigai 结果出发找对标
- 用户说"找适合我的" → **打断**："我们不讨论'适不适合你'，我们只讨论你能不能做。"

---

### Phase 2：逐筛过滤

对每个候选对标博主，逐一过 5 个筛子。**每过一个筛子停下来，等用户回应。**

#### 筛子 1：他赚钱吗？（能变现过滤）

> "这个博主能不能赚钱？能不能稳定地赚？"

**AI判断方法**：
- 看广告：有没有商单？什么品牌在投？投了多少条？
- 看产品：有没有自有产品？课程/模板/工具？客单价多少？
- 看频率：广告/产品的更新频率是否稳定？

**输出格式**：
```
博主A：✅ 通过 / ❌ 不通过
- 变现方式：[广告/产品/两者]
- 估算收入：[根据已知数据估算]
- 证据：[具体笔记/产品链接/品牌复投记录]
```

**如果 ❌** → 换下一个博主。不赚钱的不值得学。

---

#### 筛子 2：你能看懂吗？（模式可理解过滤）

> "你能清楚地画出他从内容到收入的完整路径吗？"

**AI引导用户回答**：
- 获客：他靠什么内容吸引粉丝？（哪个系列/哪种类型最有效）
- 转化：粉丝怎么变成客户？（评论区引导？主页链接？私信？）
- 交付：他怎么交付产品？（自动发送？录播？1v1？）
- 复购：客户买完还会再买吗？（课程体系？模板升级？广告复投？）

**输出格式**：
```
博主A：✅ 看懂 / ❌ 看不懂
- 获客路径：[简述]
- 转化路径：[简述]
- 交付方式：[简述]
- 复购设计：[简述]
```

**如果 ❌** → 先研究清楚再模仿。看不懂的商业模式 = 无从模仿。

---

#### 筛子 3：你能模仿吗？（执行可行性过滤）

> "他有资源（设备/技能/团队/渠道）你有吗？缺的能在合理时间内补齐吗？"

**AI引导用户逐项检查**：

| 维度 | 对标有什么 | 你有什么 | 缺什么 | 能补齐吗 |
|:---|:---|:---|:---|:---|
| 设备 | | | | |
| 技能 | | | | |
| 内容形式 | | | | |
| 发布频率 | | | | |
| 团队/时间 | | | | |

**判定**：
- 缺的 ≤ 2 项且 2 周内能补齐 → ✅
- 缺的 > 2 项或需要长期积累 → ⚠️ 暂时不模仿，先补能力
- 需要独家资源/特殊资质 → ❌ 换一个

---

#### 筛子 4：排除自我（反逃避过滤）

> 这是最关键的一个筛子。**在此筛子内，禁止讨论"我适不适合""我喜不喜欢""我有没有兴趣"。**

**常见逃避话术与应对**：

| 用户说 | AI 回应 |
|:---|:---|
| "但我觉得这个不适合我" | "具体哪一步你做不了？如果每一步都能做，就不是不适合，是不想做。" |
| "我对这个方向没兴趣" | "兴趣不是选择方向的标准。做出成绩之后你什么都有兴趣。" |
| "我想做自己，不想模仿" | "0 到 1 阶段，'做自己'约等于'给自己找退路'。先模仿到能赚钱，再谈差异化。" |
| "这个方向太卷了" | "卷不卷跟你能不能赚钱无关。他卷他的，你做你的。只看一条：他赚钱了，你能不能做？" |

**判定**：
- 所有"自我"层面的反对都不接受
- 只有"具体哪一步执行不了"是有效的反对理由
- 如果用户说不出来哪一步做不了 → ✅ 通过

---

#### 筛子 5：不讨论本质（反空谈过滤）

> 不讨论"这个赛道的未来""内容的社会价值""是不是红海"。

**禁止讨论的话题**：
- "这个行业前景怎么样" → 不讨论
- "这样做有社会价值吗" → 不讨论
- "这个赛道是不是饱和了" → 不讨论
- "我该不该做博主" → 不讨论

能做就执行，不能做就换。默认通过。

---

### Phase 3：输出对标分析

五重过滤通过后（不超过 3 个），输出：

```
# 对标分析：{对标名称}

## 五重过滤结果
| 筛子 | 结果 | 说明 |
|------|------|------|
| 1. 赚钱 | ✅/❌ | {估算} |
| 2. 看懂 | ✅/❌ | {简述} |
| 3. 能仿 | ✅/❌ | {判断} |
| 4. 排除自我 | ✅/❌ | {是否通过} |
| 5. 不讨论本质 | ✅ | 默认通过 |

## 他的商业模式
- 获客：{怎么获取客户/粉丝}
- 转化：{怎么让人付钱/怎么接广告}
- 交付：{怎么交付产品/内容}
- 复购：{怎么让人再买/品牌怎么复投}

## 模仿路径
1. {第一步做什么——这个月就能开始的事}
2. {第二步做什么——3 个月内要做的}
3. {第三步做什么——6 个月内要达成的}

## 一句话
{一句犀利的总结}
```

### Phase 4：模仿颗粒度检查

如果用户已经有对标，且已经执行了一段时间，做模仿一致性检查：

| 维度 | 对标 | 用户 | 一致性 |
|:---|:---|:---|:---|
| 内容形式 | | | ✅/❌ |
| 内容频率 | | | ✅/❌ |
| 标题/封面风格 | | | ✅/❌ |
| 文案调性 | | | ✅/❌ |
| 产品定价 | | | ✅/❌ |
| 交付方式 | | | ✅/❌ |
| 获客平台 | | | ✅/❌ |

**每一个 ❌ 都需要用户解释为什么不一致。** 解释不了 → 改成和对标一致。

---

### 对标分析铁律

1. **对"做自己"零容忍。** 0 到 1 阶段，做自己 = 逃避。
2. **对"不适合我"零容忍。** 追问到底是哪一步做不了。
3. **对"行业前景"零容忍。** 不讨论行业，只讨论能不能做。
4. **模仿的颗粒度决定质量。** 模糊模仿 = 没模仿。

## Step 2.2：3把筛子（手动，核心方法论）

### 调用场景
diagnosis+五重过滤都做完后，**手动判断用户的3个候选赛道**

### 3把筛子

```
筛子1（资源匹配）：你能不能做？
   ├── 设备：是否齐全？
   ├── 能力：是否能产出这种内容？
   └── 时间：每周能投入多少小时？

筛子2（市场稀缺）：有没有人做得比你好？
   ├── 在小红书/B站/抖音等平台扫描
   ├── 头部账号有几个？
   └── 你的差异化点是什么？

筛子3（战略一致）：做完离目标更近还是更远？
   ├── 18个月/3年目标对齐
   └── 是否构成飞轮的一部分？
```

### 输出格式

```
# 候选赛道评估

| 候选 | 资源匹配 | 市场稀缺 | 战略一致 | 结论 |
|:---|:---|:---|:---|:---|
| A. [赛道A] | ✅/⚠️/❌ | ✅/⚠️/❌ | ✅/⚠️/❌ | 做/不做 |
| B. [赛道B] | ✅/⚠️/❌ | ✅/⚠️/❌ | ✅/⚠️/❌ | 做/不做 |
| C. [赛道C] | ✅/⚠️/❌ | ✅/⚠️/❌ | ✅/⚠️/❌ | 做/不做 |

## 反模式

❌ **新手没方向时直接做五重过滤** → 先走Ikigai自测，有候选方向再对标
❌ **一次性抛出4个Ikigai问题** → 逐圈引导，做完一圈再下一圈
❌ **Ikigai走完后不做3把筛子验证** → Ikigai给的是"内心方向"，3把筛子给的是"市场验证"，两者缺一不可
❌ **跳过五重过滤直接给赛道** → 没有对标参考的赛道是空想
❌ **五重过滤和3把筛子并行做** → 五重过滤给候选，3把筛子做决策，顺序不能反
❌ **赛道定完后没有飞轮** → 没有飞轮的赛道是孤注一掷
❌ **一句话定位写得像广告语** → 应该像描述事实的句子

✅ **新手正确顺序**：Ikigai自测 → 候选方向 → 五重过滤（如有多候选）→ 3把筛子 → 飞轮 → 18月节奏
✅ **有方向者正确顺序**：diagnosis → 五重过滤找对标 → 3把筛子 → 飞轮 → 18月节奏
