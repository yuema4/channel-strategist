## 它做了什么

频道战略师是一个 AI Skill，帮你回答内容创作者的 3 个核心问题：

| 你问 | 它答 | 用什么 |
|:---|:---|:---|
| 我适合做什么方向？ | 从零找到你的频道定位 | **Ikigai 四圈自测** |
| 我的账号有什么问题？ | 给商业模式打分 | **七项检验**（0-70 分，内置）|
| 我该学谁、避谁？ | 找到对标博主 | **五重过滤找对标**（内置）|
| 这个选题值不值得做？ | 过筛子判定 | **3 把筛子** |
| 接下来 30 天拍什么？ | 排内容清单 | **18 月节奏 + 30 天清单** |

**它不是内容执行助手**——不写脚本、不起标题、不检查 AI 味。它只做一件事：**帮你做对战略决策**。

## 核心方法论

| # | 方法论 | 干什么用 |
|:---|:---|:---|
| 0 | **Ikigai 四圈** | 新手没方向时，用四圈交叉找到定位 |
| 1 | **3 把筛子** | 判断一个选题/方向做不做 |
| 2 | **飞轮模型** | 理解内容如何变收入、收入如何反哺内容 |
| 3 | **七项检验** | 给商业模式打 0-70 分 |
| 4 | **5:2:2:1 配比** | 30 天内容不跑偏 |

---

## 两条路径

```
新手（无方向）：
  Ikigai 四圈自测 → 中心方向 → 3 把筛子 → 飞轮 → 排节奏

有账号（优化方向）：
  七项检验 → 找对标 → 3 把筛子 → 飞轮 → 排节奏 → 复盘
```

## 安装

### 方式一：复制到 Claude Code skills 目录

```bash
git clone https://github.com/你的用户名/channel-strategist.git
cp -r channel-strategist ~/.claude/skills/
```

### 方式二：手动安装

1. 下载本仓库
2. 将 `SKILL.md` 和 `modules/` 和 `templates/` 放到 `~/.claude/skills/channel-strategist/`
3. 重新加载 Claude Code

## 触发方式

加载后，说以下任一关键词自动触发：

| 你说什么 | 它调什么 |
|:---|:---|

| "这周数据怎么样" | 周复盘 |
